## 项目简介

**TuxiCloud** 是一款专为个人和团队设计的图片搜索与协作平台，涵盖公共图库、个人空间和团队空间三大核心场景。无论你是个人创作者还是团队协作者，TuxiCloud 都能提供强大支持。

## 主要功能

- **多样化图库管理**  
  - 公共图库：访问丰富的图片资源。
  - 个人空间：创建和管理私人图库。
  - 团队空间：基于 RBAC 权限管理的协作空间，支持素材共享与协同编辑。

- **智能图片处理**  
  - AI 扩图：智能扩展图片内容。
  - 以图搜图：通过图片快速查找相似内容。
  - 颜色搜图：根据颜色精准搜索图片。

- **高效管理与分享**  
  - 批量管理：轻松处理大量图片。
  - 分享功能：一键分享图片或图库链接。
